
# 项目简介

这是一个纯前端的把 Minecraft Java版的 .litematic 文件转换到 .nbt 的项目

## 📋 功能特性

- 一键拖放转换
- 不限制大小
- 受浏览器性能限制

## 🚀 快速开始

### 安装

```bash
git clone https://github.com/rukiroki/litematic-to-nbt-web
npm install
npm run build
```

### 使用

1. 拖放litematic文件到[页面上](https://rukiroki.github.io/litematic-to-nbt/)
2. 等待转换
3. 下载在转换完成的nbt文件
